c프로그램의 경우 소스파일 이름(.c)과 관계없이 원하는 이름의 실행 프로그램(.exe) 생성이 가능하지만
java에서는 프로그래머가 명명한 클래스 이름으로 묶어주고 확장자 명은 .java이여야 한다.

class : 데이터를 크게 묶은 하나의 단위 (객체)

명령어 인수와 모니터 출력
C언어에서 출력을 하기 위해선 printf 를 사용하지만 
자바에서는 system.out 클래스의 
system.out.print(출력)
system.out.println(출력 후 줄변경)
system.out.printf(c 처럼 사용가능)
를 이용한다.

자바 프로그램의 키보드 입력과 문자 코드

c프로그램의 키보드 입력 방법 중의 하나는 getchar이다.(문자 하나 입력)
x = getchar(); (a를 입력시 자동으로 아스키코드값으로 변환되어 97로 저장됨)

자바 프로그램의 가장 기본적인 키보드 입력 방법은 system.in.read() 메소드(함수)이다.
x = system.in.read(); (a를 입력시 자동으로 아스키코드값으로 변환되어 97로 저장됨)

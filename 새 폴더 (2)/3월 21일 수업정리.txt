묵시적●명시적 형변환

int += float 형태의 문장에서는 작은 타입으로의 묵시적 형 변환이 이루어진다.

넘치는 부분은 잘리고 그렇지 않다면 그대로 들어간다.

메소드 중복 정의

c언어에서는 모든 함수 이름이 달라야한다.
java에서는 이름이 같은 함수가 존재할 수 있지만 매개변수의 이름이 달라야 한다.
호출시 매개변수가 일치하는 함수를 호출한다.

묵시적 형변환이 가능한 함수가 두 가지 이상일 경우
int int인 경우
float float
int float 
float int
세 가지 경우로 모두 형변환이 가능하다.
이럴경우에는 형변환이 일어나지 않는다.

중첩 클래스

c에서는 큰 구조체를 여러개의 작은 구조체들로 구성할 수 있다.
java에서도 하나의 클래스를 여러개의 작은 클래스들로 구성할 수 있다.

외부 클래스에 대한 객체를 생성하더라도 내포된 클래스의 객체는 생성되지 않는다.

중첩 클래스 범위 규칙

this : 메소드 안에서 객체를 지칭하는데 사용됨

클래스 안에 클래스와 이름이 같은 메소드를 생성자라고 하고 객체를 생성할때 자동으로 호출된다.
예)
class Add{
    int total;

    Add(int total){
        this.total = total ;
    }
}
